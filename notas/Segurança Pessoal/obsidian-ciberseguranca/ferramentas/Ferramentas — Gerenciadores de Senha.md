## Bitwarden
- **Tipo:** Cloud / Open Source / Gratuito
- **Plataformas:** Web, Windows, macOS, Linux, Android, iOS, extensões de navegador
- **Destaques:** Auditado por terceiros, suporta Passkeys, exportação fácil, self-hosting possível
- **Link:** https://bitwarden.com
- **Quando usar:** Uso geral — melhor equilíbrio entre conveniência e segurança

## KeePassXC
- **Tipo:** Offline / Open Source / Gratuito
- **Plataformas:** Windows, macOS, Linux (desktop only)
- **Destaques:** Banco de dados `.kdbx` totalmente local, sem dependência de servidor, integra com YubiKey
- **Sincronização:** Manual (Syncthing, Dropbox, etc.)
- **Quando usar:** Quem não quer nenhum dado na nuvem

## 1Password
- **Tipo:** Cloud / Pago (~US$3/mês)
- **Plataformas:** Todas
- **Destaques:** Travel Mode (oculta cofres ao cruzar fronteiras), Watchtower (monitora vazamentos), interface polida
- **Quando usar:** Usuários corporativos ou quem valoriza experiência premium

## Proton Pass
- **Tipo:** Cloud / Open Source / Freemium
- **Plataformas:** Todas
- **Destaques:** Integrado ao ecossistema Proton, aliases de e-mail integrados
- **Quando usar:** Usuários do ecossistema ProtonMail
